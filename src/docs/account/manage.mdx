---
title: Gestion de votre compte utilisateur
---

Nous vous proposons une interface unifiée qui vous permettra de gérer toutes les informations concernant votre compte.

Rendez-vous sur [Mon compte Tenoco](https://myaccount.tenoco.net/) pour accéder à cette interface.

Connectez-vous en utilisant le bouton prévu à cet effet en haut à droite.

Une fois connecté, vous aurez accès aux sections suivantes.

## Informations personnelles

Retrouvez dans cette section toutes les informations vous concernant.

Il vous est possible de modifier ces informations en cliquant dessus directement.

## Services

Cette section liste l'ensemble des services auquel votre compte vous donne accès.

## Sécurité

Cette section vous permet de gérer tous les aspects liés à la sécurité de votre compte Tenoco.

Ici, vous serez en mesure de modifier votre mot de passe ainsi que de générer des mots de passe d'application.

Un mot de passe d'application peut être utile dans le cas où vous utilisez un client qui ne prend pas en charge l'authentification moderne.
Un mot de passe vous sera alors généré et affiché qu'une seule fois, ce qui vous permettra de configurer le client en question.
Vous pouvez en générer autant que vous avez de client.
Il est également possible de lui définir une date d'expiration.
