---
title: Configurer son client pour Tenoco Chat
---

Après avoir choisi un client, il vous faudra entrer les informations suivantes :

- Serveur Matrix / Homeserver : `matrix.tenoco.net`

- Nom d'utilisateur : votre identifiant Tenoco

- Mot de passe : un mot de passe d'application que vous avez généré depuis MyAccount.

Si le serveur à utiliser ne vous est pas demandé, entrez `@Nom d'utilisateur:tenoco.net` dans le champ `Nom d'utilisateur` en remplaçant `Nom d'utilisateur` par votre identifiant Tenoco.
