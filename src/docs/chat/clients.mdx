---
title: Clients pour utiliser Tenoco Chat
---

Pour utiliser Tenoco Chat, vous pouvez utiliser ces différents clients en fonction de l'appareil sur lequel vous vous trouvez.

## Client web

Nous mettons à disposition une instance de Element Web accessible à cette adresse : [chat.tenoco.net](https://chat.tenoco.net/).

## Client Android

Vous pouvez utiliser [Element pour Android](https://play.google.com/store/apps/details?id=im.vector.app&hl=fr).

## Client iOS

Vous pouvez utiliser [Element pour iOS](https://apps.apple.com/fr/app/riot-im/id1083446067).
