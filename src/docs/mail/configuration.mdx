---
title: Configuration de son client de messagerie
---

## Courrier entrant

Le protocole utilisé pour recevoir du courrier électronique est **IMAP**.

Voici les informations que vous devrez fournir à votre client de messagerie afin qu'il puisse être en mesure de récupérer du courriel :

- Serveur : **imap.tenoco.net**
- Port : **993**
- SSL : **SSL/TLS**

Configuration alternative :

- Serveur : **imap.tenoco.net**
- Port : **143**
- SSL : **STARTTLS**

## Courrier sortant

Le protocole utilisé pour envoyer du courrier électronique est **SMTP**.

Voici les informations que vous devrez fournir à votre client de messagerie afin qu'il puisse être en mesure d'envoyer du courriel :

- Serveur : **smtp.tenoco.net**
- Port : **465**
- SSL : **SSL/TLS**

Configuration alternative :

- Serveur : **smtp.tenoco.net**
- Port : **587**
- SSL : **STARTTLS**
